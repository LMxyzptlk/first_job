## Tree utility

`tree.py` prints the directory structure of a folder. Run:

```bash
python tree.py path/to/folder
```

If no path is given, it defaults to the current directory.

## File manager

`file_manager.py` offers helper functions for copying and deleting files as well as
a small CLI to get size or file count of a path. Example:

```bash
python file_manager.py size path/to/file_or_directory
python file_manager.py count path/to/file_or_directory
```
